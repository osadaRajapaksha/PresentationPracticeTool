# 🎤 Presentation Practice Tool

A lightweight and interactive tool designed to help users improve their presentation and public speaking skills. The application assists with presentation rehearsal by providing timing support, structured guidance, and simulated audience interaction to build confidence and enhance delivery.

---

## 🚀 Features

* ⏱️ **Presentation Timer** – Monitor presentation duration in real-time.
* 📝 **Script Support** – Load and follow presentation scripts during practice sessions.
* 🎯 **Audience Feedback Simulation** – Receive randomized questions and prompts to mimic real audience interaction.
* 📊 **Performance Analysis** – Track pacing, identify filler words, and evaluate presentation flow.
* ⚙️ **Customizable Settings** – Configure timing limits, feedback frequency, and difficulty levels.
* 🎓 **Skill Development** – Improve confidence, communication, and presentation effectiveness.

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/osadaRajapaksha/PresentationPracticeTool.git
cd PresentationPracticeTool
```

### Install Dependencies

Ensure Python 3.8 or higher is installed, then run:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the application:

```bash
python main.py
```

### Optional Parameters

#### Practice with a Script

```bash
python main.py --script my_talk.txt
```

#### Set a Presentation Time Limit

```bash
python main.py --time 10
```
